## 🩺 Breast Cancer Detection using Logistic Regression using machine learning

## 📌 Project Overview
This project focuses on predicting whether a tumor is **malignant** or **benign** using machine learning.  
We use **Logistic Regression**, a popular classification algorithm.

---

## 🎯 Objective
- Build a classification model
- Analyze medical dataset
- Predict cancer diagnosis accurately

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## 📊 Dataset
- Breast Cancer dataset (from sklearn / UCI)
- Features include:
  - Radius
  - Texture
  - Perimeter
  - Area
  - Smoothness

---

## ⚙️ Steps Performed
1. Data loading and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature selection  
4. Model building using Logistic Regression  
5. Model evaluation (accuracy, confusion matrix)

---

## 📈 Results
- Achieved good accuracy in classification
- Model can effectively distinguish between benign and malignant tumors
